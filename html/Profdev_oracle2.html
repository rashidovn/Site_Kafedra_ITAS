Цель курса:

Получить знания, необходимые для выполнения задач администратора базы данных
Получить практические навыки администрирования СУБД Oracle во время лабораторных работ, составляющих более половины курса

Дополнительной целью курса является подготовка к сертификационному экзамену 1Z0-042 Oracle Database 10g: Administration I (Совместно с курсом Основы администрирования СУБД Oracle, уровень 1)

По окончании курса вы сможете:

  • управлять сетевыми соединениями с базой данных;
  • выполнять проактивное администрирование;
  • выполнять резервное копирование и восстановление базы данных;
  • управлять безопасностью базы данных

Рекомендованные предыдущие курсы:
Основы администрирования СУБД Oracle, уровень 1

Содержание курса:

  • Управление сетевыми подключениями к базе данных:
    1. Архитектура Oracle Network Environment
    2. Выделенный сервер и разделяемый сервер
    3. Управление слушателями
    4. Управление сервисами базы данных
    5. Утилиты для управления сетевой конфигурацией
  • Проактивное управление базой данных:
    1. Оптимизатор. Сбор статистики оптимизатора
    2. Управление Automatic Workload Repository
    3. Использование Automatic Database Diagnostic Monitor
  • Управление производительностью:
    1. Использование Enterprise Manager для мониторинга производительности
    2. Повышение производительности запросов с помощью SQL adviser и access adviser
    3. Использование автоматического управления памятью
    4. Использование memory adviser при ручном управлении памятью
  • Резервное копирование и восстановление:
    1. Основы резервного копирования Oracle
    2. Резервное копирование и восстановление, управляемое пользователем
    3. Резервное копирование и восстановление с помощью RMAN
  • Использование технологии Flashback:
    1. Основные концепции и возможность технологии Flashback
    2. Восстановление базы данных на определенный момент времени в прошлом
    3. Восстановление удаленной таблицы
    4. Запросы к базе данных по состоянию на определенный момент времени в прошлом
    5. Просмотр истории транзакций
  • Миграция данных:
    1. Использование DataPump
    2. Использование SQL Loader
    3. Внешние таблицы

Срок обучения: 72 часа
Форма обучения: вечерняя, без отрыва от работы
Стоимость обучения: 5 000 рублей
