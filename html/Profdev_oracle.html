«ОСНОВЫ АДМИНИСТРИРОВАНИЯ СУБД ORACLE, УРОВЕНЬ 1»

Цель курса:

Получить необходимые знания и навыки для создания и сопровождения базы данных Oracle.
Дополнительной целью курса является подготовка к сертификационному экзамену 1Z0-042 Oracle Database 10g: Administration I

По окончании курса вы сможете:

  • создавать базы данных;
  • выполнять функции администратора базы данных;
  • управлять пользователями и ресурсами базы данных

Содержание курса:

  • Установка и создание базы данных
    1. Архитектура СУБД Oracle
    2. Установка программного обеспечения СУБД Oracle
    3. Создание базы данных Oracle
  • Управление базой данных
    1. Управление экземпляром СУБД Oracle
    2. Словарь данных
    3. Управляющие файлы
    4. Управление журнальными файлами
    5. Управление табличными пространствами и файлами данных
    6. Структуры хранения СУБД Oracle
    7. Управление данными отмены
    8. Управление пользователями
    9. Технология Automatic Storage Management (ASM)
  • Управление данными в СУБД Oracle
    1. Управление объектами схемы
    2. Управление целостностью
    3. Управление одновременной работой

Срок обучения: 72 часа
Форма обучения: вечерняя, без отрыва от работы
